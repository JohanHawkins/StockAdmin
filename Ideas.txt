# Sistema de Gestión de Inventario Web

## Objetivo del Proyecto

Construir una aplicación web moderna para administrar inventario de manera sencilla y profesional.

El sistema permitirá:
- Iniciar sesión
- Administrar productos
- Controlar stock
- Registrar entradas y salidas
- Visualizar información básica del inventario

---

# Tecnologías del Proyecto

## Frontend
Angular

Responsable de:
- Interfaces visuales
- Formularios
- Dashboard
- Consumo de API

---

## Backend
ASP.NET Core

Responsable de:
- Lógica del negocio
- Validaciones
- API REST
- Autenticación

---

## Base de Datos
PostgreSQL

Responsable de:
- Almacenar usuarios
- Productos
- Movimientos
- Categorías

# Flujo Básico del Sistema

1. Usuario inicia sesión
2. Ingresa al dashboard
3. Administra productos (crear, editar, activar/desactivar, eliminar)
4. Registra entradas o salidas
5. El sistema actualiza el stock automáticamente
6. Visualiza estadísticas en el dashboard
