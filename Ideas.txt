## Objetivo del Proyecto

Construir una aplicación web moderna para administrar inventario de manera sencilla y profesional.

---

# Tecnologías

- **Frontend:** Angular
- **Backend:** ASP.NET Core (futuro)
- **Base de Datos:** PostgreSQL (futuro)

---

# Permisos por Rol

| Funcionalidad | Admin | Usuario |
|---------------|-------|---------|
| Dashboard | CRUD completo | Solo lectura |
| Productos | Crear, Editar, Eliminar, Activar/Desactivar | Solo lectura |
| Categorías | Crear, Editar, Eliminar | Solo lectura |
| Movimientos | Crear, Filtros | Crear, Filtros |

# Estado del Proyecto

## Completado (91%)

### Login ✅

- Guard de rutas protegidas

### Dashboard ✅
- Total de productos, categorías, movimientos

- Producto con mayor stock y más costoso

- Formato de moneda con separadores

### Categorías ✅

### Movimientos ✅

- Filtros por fecha, tipo y producto

### UI/UX ✅

- Responsive (móvil)

### Modelos de Datos ✅

# Flujo del Sistema

1. Usuario inicia sesión (admin o usuario)
2. Ingresa al dashboard según su rol
3. Administra productos (según permisos)
4. Registra entradas o salidas
5. El sistema actualiza el stock automáticamente
6. Visualiza estadísticas en el dashboard
